1. Define your source

It's possible to use the TurtleGrid with the HTML, XML or JSON. The snippet below is only for HTML. If you want to use XML or JSON please have a look at the sources option.

<div id="#container">
    <div><img src="your path image"></div>
    <div><img src="your path image"></div>
    <div>
        Here, add all HTML contents you want
        (images, forms, textes, videos, canevas, ...)
    </div>
    ...
</div>

2. Scripts & Stylesheets

To define which files you need, you have to ask this question: "Do you use the TurtleSlider?"

Add these sheets into your project directory and place references into your HTML page.

<link rel="css/tgrid.min.css" rel="stylesheet">
<script src="js/jquery-min.js"></script>
<script src="js/tgrid-jquery.min.js"></script>

Add these sheets into your project directory and place references into your HTML page.

<link rel="css/tgrid-nos.min.css" rel="stylesheet">
<script src="js/jquery-min.js"></script>
<script src="js/tgrid-nos-jquery.min.js"></script>
jQuery 1.11+ is requierd

3. Initialization of the grid

The framework needs to know which container will be the parent of the grid boxes! Define it into an external javascript file or in the <head> section of your HTML page like the following example. To get information about options just click here.

$(document).ready(function() {
    $('#container').tGrid({ 
        // your options 
    });
});

2. How to define caption's content

HTML

It's possible to set caption on all boxes. You just need to add a little snippet in the HTML code of your boxes and to define which effect, theme and animation you want.

Add the .caption container into your boxes (after the main element). Define your own content!

<div>
    <img src="your image path">
    <div class="caption">
        <h3>Title</h3>
        <div class="description">
           <p>...</p>
        </div>
        <button>...</button>
    </div>
</div>

For animation and simplicity reasons, the first element must be the main content (image, video, caneva, ...)!

Other sources

It's also possible to define captions' content with JSON and XML. Have a look at the sources option.

3. Text animation

To activate it, you can define it directly into the framework options with the caption option set text_animation to true/false, you can also define it per size. Have a look at the caption option.

More ways
There's two other ways to activate the text animation effect:
  • Add the tg-cc-animation in the boxClasses option.
  • Define it directly in the HTML (just add the tg-cc-animation at the same place as the caption effect class - see how to define it).

4. Themes

To define a theme, you can do it directly into the framework options with the caption option. Set theme with your value, you can also define a theme per size. Have a look at the caption option.

More ways
There's two other ways to define the theme's caption:
  • Add the tg-ct-[dark/light] in the boxClasses option.
  • Define it directly in the HTML (just add the tg-ct-[dark/light] at the same place as the caption effect class - see how to define it).

5. Hover effects

To define a caption effect, you can do it directly into the framework options with the caption option. Set effect with your value, you can also define a caption effect per responsive size. Have a look at the caption option.

More ways
There's two other ways to define the caption's effect:
  • Add the tg-c-['your-effect'] in the boxClasses option.
  • Define it directly in the HTML (just add the tg-c-['your-effect'] at the same place as the caption effect class - see how to define it).

Arguments

Activate or not the caption's text animation.

Define caption's theme. You can use one of the existing (light or dark) or define your own class by adding a point before like '.my-class'.

Define caption's hover effect. You can use one of the existing or define your own class by adding a point at the beggining like '.my-effect'.

The function have two arguments, the element $element and the queue queue (use the last one to make alternate effect).

Sample alternate

      // - you callback  });
}

Sample not alternate

animation_custom_fct : function($element, queue) {
    $element.animate({  // - your css  }, 200, function() {
      // - you callback  });
}

Getters

Getters give you information about the framework, they could be called after the initialization of it.

settings

This getter gives you the possibility to know all options used by the framework.

$('#container').tGrid.settings;

currentSize

This getter gives you the possibility to know the current size used by the framework.

$('#container').tGrid.currentSize;
